# Simple Interest Calculator

A comprehensive tool designed to calculate simple interest based on the principal amount, annual interest rate, and time period in years.

## Formula
The formula used for calculating simple interest is:
A = P(1 + rt)

Where:
- P = Principal amount
- r = Annual interest rate
- t = Time period in years

## How to Use
Run the script `simple-interest.sh` in your terminal and follow the prompts to enter your financial variables.
